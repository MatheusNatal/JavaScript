Ex 03

Operações matemáticas utilizando variáveis

var n1 = Number.parseFloat(window.prompt('Digite um número: '))

var n2 = Number.parseFloat(window.prompt('Digite outro número: '))

var soma = n1 + n2

window.alert(`A soma entre os dois números é : ${soma}`)

Converter número para string

  • parseFloat: converte a variável para um número float
  • ${variavel} - Interpolação, mostra determinada variável na frase sem precisar concatenar.

Exemplo de concatenação

var nome = 'Matheus'

var idade = 24

var nota = 8.5
Resultado:
'O aluno ' + nome + 'tem ' + idade + 'anos e tirou a nota ' + nota'

Exemplo de interpolação

var nome = 'Matheus'

var idade = 24

var nota = 8.5
Resultado:
`O aluno ${nome} tem ${idade} anos e tirou a nota ${nota}`

Ex 04

Trabalhando com inputs de usuário

var nome = window.prompt('Qual é o seu nome?')

A linha acima abre uma janela no navegador para o usuário digita algo.

document.write(`Olá, ${nome}! Seu nome tem ${nome.length} letras.`)

A linha acima responde a janela anterior, saudando o usuário com o nome que ele colocou.

document.write(`Seu nome em maiúsculas fica: ${nome.toUpperCase()})`

A linha acima coloca a variável {nome} em maiúscula.

document.write(`Seu nome em minúsculas fica: ${nome.toLowerCase()}`)

A linha acima coloca a variável {nome} em minúscula.

Trabalhando com números

var n1 = 1501.5

n1.fixed(2)
Resultado:
'1501.50'

A linha "n1.fixed(2)" coloca 2 casas decimais após o ponto

Por padrão, na programação, o "." é equivalente a ","

n1.toFixed(2).replace('.', ',')
Resultado:
'1501,50'

A linha "n1.toFixed(2).replace('.', ',')" converte todo "." para "," e vice-versa.

Transformando número para formato de salário.

n1.toLocaleString('pt-BR', {style: 'currency', currency: 'BRL'})
Resultado:
'R$1.501,50'

Pode também trocar o 'BRL' para outros formatos, como 'USD', 'EUR'
